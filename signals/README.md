# Signals

This directory captures the upstream inputs that inform `AI Native`.

Its purpose is to detect common threads in how the industry is using AI across
the software development lifecycle, while filtering out hype and one-off noise.

## Contents

- source registry
- ontology
- scoring rubric
- evidence templates

## Workflow

1. Monitor trusted sources.
2. Capture evidence records.
3. Normalize observations into shared concept types.
4. Escalate repeated or meaningful changes into decision records.
