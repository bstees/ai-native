# Repo Rules Assets

This directory contains shared rule assets intended for consumer repositories.

## Current Asset

- [`ai-native-core-operating-rules.md`](./ai-native-core-operating-rules.md)
  provides a centrally maintained baseline for AI-agent behavior in production
  repos.

## Intended Use

This asset should be consumed as a starting point and supplemented with
repo-local details rather than rewritten from scratch in every repo.
